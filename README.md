# SysPharma: Systems Pharmacoproteomics of Mitochondrial Drug Targets in Colorectal Cancer

Code repository for:  
**"A Pharmacoproteomic Scoring Framework for Mitochondrial Drug Target Prioritization in Colorectal Cancer"**  
Tsz-Leung To — Independent Researcher, Boston, USA  
*Submitted to Molecular & Cellular Proteomics*

---

## Repository structure

```
syspharma/
├── README.md
├── requirements.txt
├── 01_prepare_dimensions.py      # Compute all 6 raw scoring dimensions per gene
├── 02_compute_zc.py              # Carbon oxidation state (ZC) from UniProt FASTA
├── 03_gene_priority_scoring.py   # Gene-level priority score (MitoCarta + all-proteome)
├── 04_pathway_priority_scoring.py# Pathway-level priority score (38 MitoPathways)
├── 05_hallmark_scoring.py        # MSigDB Hallmark pathway scoring
├── 06_drug_coregulation.py       # Pairwise Pearson correlations + permutation test
├── 07_bioplex_analysis.py        # BioPlex shared interaction partner analysis
├── 08_jacques_proteomics.py      # ONC212 NALM6 proteomics reanalysis (Fig. 3H)
├── 09_kudryashova_proteomics.py  # ML210 MEF proteomics reanalysis (Fig. 4F)
├── 10_pathway_sensitivity.py    # Dirichlet weight sensitivity analysis (Fig. S2)
├── data/                         # Input data (downloaded per instructions in README)
└── results/                      # Output CSV files (regenerated by running scripts)
```

## Input data

All input data are publicly available. Download to the `data/` directory:

| File | Source | Description |
|------|--------|-------------|
| `Human.MitoCarta3.0.xls` | https://www.broadinstitute.org/mitocarta | MitoCarta 3.0 gene annotations |
| `OmicsExpressionProteinCodingGenesTPMLogp1_23Q4.csv` | https://depmap.org | DepMap 23Q4 CRISPR Chronos scores |
| `CRISPRGeneEffect_23Q4.csv` | https://depmap.org | DepMap 23Q4 Chronos gene effect |
| `mitchell_log2fc.csv` | MassIVE MSV000089282 | Drug-proteome atlas (Mitchell 2023) |
| `mmc2_1.xlsx` | Li et al. 2021 Mol Cell (PXD024513) | Protein half-lives raw data |
| `drugmap_sites.csv` | Takahashi et al. 2024 Cell Supp Table S3 | DrugMap CRA sites |
| `slcabpp_cr.csv` | Kuljanin et al. 2021 Nat Biotechnol (PXD016553) | SLC-ABPP max competitive ratios |
| `uniprot_sequences.fasta` | https://www.uniprot.org | UniProt canonical sequences |
| `Table_S2_Jacques_et_al_accepted.xlsx` | Jacques et al. 2020 Genetics figshare 11873841 | ONC212 NALM6 proteomics |
| `kudryashova_log2fc.csv` | Kudryashova et al. 2023 Data MDPI (PRIDE PXD041327) | ML210 MEF proteomics |
| `BioPlex_HCT116_Network_5_5K_Dec_2019.tsv` | https://bioplex.hms.harvard.edu | BioPlex 3.0 HCT116 PPI network |
| `h.all.v2023.1.Hs.symbols.gmt` | MSigDB https://www.gsea-msigdb.org | Hallmark gene sets |

## Usage

```bash
pip install -r requirements.txt

# Run in order; each script reads from data/ and writes to results/
python 01_prepare_dimensions.py
python 02_compute_zc.py
python 03_gene_priority_scoring.py
python 04_pathway_priority_scoring.py
python 05_hallmark_scoring.py
python 06_drug_coregulation.py
python 07_bioplex_analysis.py
python 08_jacques_proteomics.py
python 09_kudryashova_proteomics.py
python 10_pathway_sensitivity.py
```
